## 🎬 Animation Details

### Hero Section Animations
- **Text fade-in:** Headline, tagline, buttons fade in sequentially (stagger 0.2s)
- **Fabric swatches:** Floating shapes drift down with varying speeds
- **Scroll indicator:** Pulsing down arrow to encourage scrolling

### Product Cards
- **Hover state:** Subtle shadow increase, slight scale-up
- **Texture animation:** Pattern overlay opacity changes

### Contact Form
- **Focus states:** Ring around inputs, smooth transitions
- **Submit button states:**
  - Idle: Forest green
  - Sending: Yellow with spinner animation
  - Sent: Green checkmark
- **Auto-reset:** Form clears after successful send

### Scroll Triggers
- **Navigation:** Glass blur effect kicks in after 50px scroll
- **Section reveals:** Card entrances as they come into view

## 🎨 Customization Quick Guide

Want to change something? Here's where:

| Element | Location | How |
|---------|----------|-----|
| Colors | `tailwind.config.js` | Change hex values in theme.colors |
| Fonts | `index.html` | Update Google Fonts links |
| Text Content | `src/App.jsx` | Find and replace strings |
| Images | `src/App.jsx` | Update image URLs |
| Contact Info | `src/App.jsx` | Update phone, email, hours |
| Logo | `src/App.jsx` | Change logo text or add image |
| Animations | `src/App.jsx` | GSAP settings in useEffect hooks |
